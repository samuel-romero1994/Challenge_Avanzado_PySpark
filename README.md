# Challenge_Avanzado_PySpark
## Descripción
Este proyecto forma parte de un challenge de Machine Learning utilizando PySpark para el procesamiento de datos a gran escala. El objetivo fue cargar un dataset, realizar limpieza y análisis exploratorio básico, y preparar las características para un modelo de clasificación.
Se realizaron tareas de:

-Revisión y validación de columnas con PySpark.

-Conversión del DataFrame de Spark a Pandas para estadísticas rápidas.

-Limpieza y selección de columnas relevantes.

-Generación de estadísticas descriptivas de las features.

-Tecnologías y librerías utilizadas

-Python

-PySpark

-Pandas

-NumPy
